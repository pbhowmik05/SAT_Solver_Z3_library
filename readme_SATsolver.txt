The program checks whether a given Sudoku configuration has a solution and
output one of those solutions. It uses prepositional logic to encode the problem
and solves the puzzle by using satisfiability checking functionality of Z3. This program
is implemented in python3 and needs the additional library from numpy and Z3.
The program takes an a sudoku puzzle as an input from CSV file. If the board has a solution then
it prints the output otherwise says the given puzzle has no solution.
