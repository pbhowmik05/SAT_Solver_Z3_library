The program is implemented in c and it reads a text file to solve a sudoku problem. The text file has numbers separated by comma and has nine rows and nine columns.

In this code the main function is sudoku engine which is responsible to give a the valid solution. The placement check helps the sudoku engine to find the empty and valid position. Then the route function validates the routing the numbers to the correct position.

Besides, if the given problem has data redundancies according to the sudoku rule, then the replacement check function gives the number of conflicting data and their location and terminates the program. If the given problem has no solution then the program showed that there is no solution.
